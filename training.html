STEP 07
📁 다운로드 & 폴더 준비

  3. project 폴더 안에 practice 폴더를 만듭니다
  4. STEP 1에서 내려받은 사진들을 C:\project\practice에 복사합니다

▶

⚠️
보안 경고가 뜰 수 있어요! PowerShell에서 "실행 정책" 오류가 나오면 아래 버튼을 클릭하세요.

PowerShell을 관리자 권한으로 열고 아래 명령 먼저 실행하세요:
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
그 후 다시 설치 명령을 실행합니다.

✅
버전 번호(예: 1.x.x)가 출력되면 설치 성공입니다!
PATH 환경변수에 Claude가 설치된 경로를 추가해야 합니다. 아래 명령을 실행하세요:
[Environment]::SetEnvironmentVariable("Path", $env:Path + ";$env:USERPROFILE\.localin", "User")

STEP 13
📝 PRD 작성 요청
▶

먼저 Claude에게 프로젝트 요구사항 문서(PRD)를 만들어달라고 합니다. 이 문서가 다음 단계의 설계도가 됩니다.

Java 퀴즈 게임 PRD를 quiz-prd.md 파일로 작성해줘. 프로젝트 개요: - Java 기초 학습자를 위한 4지선다 객관식 퀴즈 게임 - 브라우저에서 바로 실행되는 웹 앱 - 순수 HTML, CSS, JavaScript로 구현 (프레임워크 없음) 게임 규칙: - 4지선다 객관식 문제 - 카테고리: Java 기초 (변수/자료형, 연산자, 제어문, 메서드 등) - 총 40문제 (10문제씩 4세트로 랜덤 출제) - 정답/오답 즉시 피드백 (해설 포함) - 최종 점수 및 순위 기록 (localStorage에 저장) 필수 화면: - 시작 화면 (닉네임 입력) - 문제 풀이 화면 (진행률 표시) - 결과 화면 (점수, 순위, 다시 풀기 버튼) - 순위표 화면 (TOP 10) 구조가 잘 정리된 PRD 문서로 파일에 저장해줘.
📋
PRD(Product Requirements Document)란 제품 요구사항 문서입니다. 개발 전에 "무엇을 만들지"를 명확히 정의해 두면 AI가 더 정확하게 구현합니다.
✓
STEP 14
🎮 퀴즈 앱 구현
▶

PRD가 만들어지면 이제 실제 앱을 만들어달라고 합니다. Claude가 스스로 단계를 나누고 순서대로 실행합니다.

quiz-prd.md를 읽고, 이 PRD대로 완성된 웹 앱을 만들어줘. 단계를 스스로 계획하고, 순서대로 실행하고, 각 단계가 끝나면 간단히 보고해줘.
🎓
지켜봐야 할 것:
  • Claude가 파일을 어떤 순서로 만드는지 확인
  • 각 단계 보고를 읽으며 진행 상황 파악
  • 완성 후 브라우저에서 quiz.html 파일 열기
✓
STEP 15
🗂️ 바탕화면 파일 정리 (PARA)
▶

Claude Code에게 바탕화면 파일을 자동으로 분류·정리시켜 봅니다. 먼저 내 바탕화면 경로를 확인하세요.

📍
내 바탕화면 주소 확인하기

Claude Code에서 아래 명령을 입력하면 바탕화면 경로를 알 수 있습니다.

내 바탕화면(Desktop) 폴더의 전체 경로를 알려줘

바탕화면 경로를 확인했으면, 이제 PARA 기법으로 정리를 시작합니다.

📚
PARA 기법이란?

모든 파일을 4가지 폴더로 분류하는 정리법입니다.

  • Projects — 현재 진행 중인 프로젝트 파일
  • Areas — 지속적으로 관리하는 영역 (수업자료, 행정 등)
  • Resources — 참고자료, 템플릿, 이미지 등
  • Archives — 완료되었거나 당장 안 쓰는 파일
내 바탕화면 폴더를 분석해서 PARA 기법에 따라 정리해줘. 1. 먼저 바탕화면에 어떤 파일들이 있는지 목록을 보여줘 2. 각 파일을 Projects / Areas / Resources / Archives 중 어디에 넣을지 분류표를 만들어줘 3. 분류표를 보여주고 내가 확인한 뒤에 실제로 폴더를 만들고 파일을 옮겨줘 4. 바로 옮기지 말고 반드시 내 확인을 받은 후에 실행해
⚠️
주의: "반드시 내 확인을 받은 후에 실행해"를 꼭 포함하세요. 이 문구가 없으면 Claude가 바로 파일을 옮길 수 있습니다.

STEP 16
⌨️ 핵심 슬래시 명령어
▶

Claude Code 대화 중 /로 시작하는 명령어로 다양한 기능을 활용할 수 있습니다.

/context
현재 대화에서 사용 중인 파일/컨텍스트 정보를 보여줍니다.
📋 회의록처럼 — 지금까지 공유된 내용을 정리해줘요
/export
현재 대화 내용을 파일로 내보냅니다.
💾 다른 이름으로 저장처럼 — 대화를 기록으로 남겨요
/resume
이전 대화를 이어서 계속합니다. 먼저 Ctrl+C로 Claude를 종료한 뒤, 다시 claude를 실행하고 /resume을 입력하세요.
📚 책갈피처럼 — 마지막으로 읽던 곳을 기억해줘요
/compact
긴 대화를 요약해 컨텍스트를 압축합니다. 대화가 길어질 때 사용하세요.
🗜️ 파일 압축처럼 — 핵심만 남겨서 가볍게
/agents
여러 AI 에이전트를 동시에 실행해 복잡한 작업을 병렬로 처리합니다.
👥 팀 회의처럼 — 여러 전문가가 동시에 일해줘요
